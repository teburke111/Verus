## Overview 
The **Deepfake & Synthetic Media Detector** is a cloud-native, Kubernetes-based application designed to detect AI-generated or manipulated content across text, audio, image, and video. The system integrates a React frontend, a Flask
API backend, and distributed microservices for preprocessing, inference, persistence, and authentication.
